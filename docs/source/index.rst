Welcome to PondPy's documentation!
==================================

.. toctree::
   :maxdepth: 1
   :caption: Contents:

PondPy is a package built to aid in the design of low-slope roof bays
for ponding stability and impounded rain loading. It utilizes a series
of purpose-built finite element and ponding analysis classes and methods
to be as intuitive as possible for the modern engineer.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <Installation>` the project and an
:ref:`example implementation <Example Implementation>`.
